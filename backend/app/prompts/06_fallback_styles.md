# Fallback 风格配置

当 Meooo skill 不可用时，使用以下内置降级风格配置。

## minimal（默认）

- **颜色**
  - 主色：`#171717`
  - 次色：`#52525b`
  - 背景：`#ffffff`
  - 强调色：`#6366f1`
- **字体**
  - 字体族：`'Inter', system-ui, sans-serif`
  - 标题字重：`600`
  - 正文字号：`14px`
- **圆角**：`0.375rem`
- **阴影**
  - 小阴影：`0 1px 2px 0 rgb(0 0 0 / 0.05)`
- **反模式**
  - 不要用紫渐变
  - 不要用 Inter 之外的通用字体
  - 避免千篇一律的卡片设计

## dark

- **颜色**
  - 主色：`#e2e8f0`
  - 次色：`#94a3b8`
  - 背景：`#0a0a0a`
  - 强调色：`#818cf8`
- **字体**
  - 字体族：`'Inter', system-ui, sans-serif`
  - 标题字重：`700`
  - 正文字号：`14px`
- **圆角**：`0.5rem`
- **阴影**
  - 小阴影：`0 1px 3px 0 rgb(255 255 255 / 0.1)`
- **反模式**
  - 不要用纯白文字
  - 避免使用蓝色系配色
  - 避免过亮的背景

## glassmorphism

- **颜色**
  - 主色：`#6366f1`
  - 次色：`#8b5cf6`
  - 背景：`rgba(255,255,255,0.1)`
  - 强调色：`#f59e0b`
- **字体**
  - 字体族：`'Inter', system-ui, sans-serif`
  - 标题字重：`600`
  - 正文字号：`14px`
- **圆角**：`1rem`
- **阴影**
  - 小阴影：`0 8px 32px 0 rgb(31 38 135 / 0.37)`
- **反模式**
  - 不要用实色背景
  - 避免无透明度的元素
  - 不要过度使用模糊效果

## vibrant

- **颜色**
  - 主色：`#f59e0b`
  - 次色：`#f97316`
  - 背景：`#fffbeb`
  - 强调色：`#ef4444`
- **字体**
  - 字体族：`'Inter', system-ui, sans-serif`
  - 标题字重：`700`
  - 正文字号：`14px`
- **圆角**：`0.75rem`
- **阴影**
  - 小阴影：`0 4px 6px -1px rgb(245 158 11 / 0.3)`
- **反模式**
  - 不要用灰色系
  - 避免过度设计
  - 不要使用小号字体
