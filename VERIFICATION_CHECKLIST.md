# 功能验证检查清单

## 🎯 验证目标

确保 WebContainer 的 React 模板功能在前后端完全正常工作，用户可以顺利创建和运行 React 项目。

## 📋 验证前准备

### 环境要求
- [ ] 后端服务已启动 (`python -m uvicorn app.main:app --reload`)
- [ ] 前端服务已启动 (`npm run dev`)
- [ ] 网络连接正常
- [ ] 浏览器支持现代 JavaScript

### 依赖检查
- [ ] Node.js 和 npm 已安装
- [ ] Python 3.8+ 已安装
- [ ] 所有依赖包已安装

## 🔧 后端功能验证

### API 接口测试

#### 1. 健康检查
- [ ] `GET /api/health` 返回 `{"status": "ok"}`
- [ ] 服务响应正常

#### 2. 模板列表 API
- [ ] `GET /api/webcontainer/templates` 返回 7 个模板
- [ ] 每个模板包含 name, description, features
- [ ] 响应格式正确

#### 3. 模板创建 API
- [ ] `POST /api/webcontainer/projects/template` 成功创建项目
- [ ] 返回项目路径和文件列表
- [ ] 支持所有 7 个模板类型

#### 4. 项目状态 API
- [ ] `GET /api/webcontainer/projects/{id}/{v}/status` 返回正确状态
- [ ] 包含文件列表和依赖信息

#### 5. 依赖安装 API
- [ ] `POST /api/webcontainer/projects/{id}/{v}/install` 成功安装依赖
- [ ] 返回安装结果

#### 6. 服务器启动 API
- [ ] `POST /api/webcontainer/projects/{id}/{v}/start` 成功启动服务器
- [ ] 返回服务器 URL 和端口
